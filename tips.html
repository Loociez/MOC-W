Chat Commands:

Mirage online classic has many essential commands. /room [name] [password] - Join/leave a player public/private chatroom. /help - See this list of commands again. /description [text] - Display or change description seen when players target you. /fps - See current frames per second performance. /redraw - Redraw the game screen if it was never fully drawn. /clear - Clear all received game chat messages. 2 /screenshot - Downloads a screenshot of the game screen. (This does not include players or NPCS) - /who - Lists all players online. (There is also an alternate version of this) /whoroom - Lists all players in player public/private chatroom. /where - Shows your current location. (Map ID and coordinates) /ping - See current server latency. /played - See how much time you have played. /chests - See how many chests you have opened. /born - See when your character was created. /uptime - See how long the server has been online. /motd - View current message of the day. 3 /ignore [name] - Ingores messages from a player or shows who is ignored without inputting a name /listen name - Listens to messages from a player. /join - Joins a pending battle royale. /withdraw - Withdraws from a pending battle royale. /register code - Register a new player code. /exit - Exits a player housing area. (Dragonia only) /quit - Quit game and go back to login screen. - /pvp - Join a no-drop PVP event for a token. You are unable to see other players name and sprite. /joindrops - Join a capture crendale with drops for a token. It is recommended to bring a guildmate with you. /joinnodrops - This event rarely occurs /hallway - A disorientating PVP event with a cost to join /stadium - No drop PVP event where the last alive wins. /whopit - Shows the players map location in the dungeon Pit. Specific commands can only be used by patreon supporters /hue - Changes your sprite colour /sprite - Changes your sprite /brightness - Changes your sprite brightness /contrast - Changes your sprite contrast /whoalt - Shows players on the same IP /requestaccess - Allows you to request /editmap permissions to edit your guild hall. Only specific people are allowed to use this. There are also more commands for utility or comedic purposes. /mirror - Displays who has the highest and lowest ping /whodelay - Can be used to show chat cooldown for different players /beastly - Find out yourself! /dj - Lets vibe together
